
This project uses HTML, CSS and JavaScript to create an advanced web-based audio player.
What's more, Web Audio API are used in this project.

The main functionalities of the app includes:
1) Select an audio file and play online. When playing, the audio will be visualised by both waveform and spectrum
2) Have control of the 10 audio effects and its parameters to the audio, which includes Stereo Pan, Delay / Echo, Reverb, Distortion / Overdrive, Compressor, Phaser, Flanger, Chorus, Tremolo and Vibrato.
3) Change the pitch of the audio file
4) Have control of the 8 filters and its parameters to the audio, which includes
low-pass, high-pass, band-pass, Notch, low-shelf, high-shelf, peaking and all-pass.
5) Save/download the edited file
